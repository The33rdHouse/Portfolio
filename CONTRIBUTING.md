# Contributing

This repository is maintained as a public portfolio for The 33rd House.

## Contribution Rules

Before committing or opening a pull request:

1. Keep content public-safe.
2. Do not include credentials, keys, tokens, passwords, or private URLs.
3. Do not include personal identity documents, legal records, private emails, phone numbers, or addresses.
4. Use placeholders in examples.
5. Keep project descriptions factual, concise, and professional.

## Recommended Workflow

1. Create a branch for material changes.
2. Review changed files before committing.
3. Confirm no `.env` or private files are included.
4. Open a pull request for review where possible.
5. Merge only after checking for sensitive data.

## Public Content Standard

Good public content:

- Explains the project clearly.
- Avoids private operational details.
- Uses professional language.
- Documents what the repo is for.
- Keeps security and privacy intact.

Not acceptable:

- Raw credentials or secrets
- Private screenshots
- Internal disputes or legal material
- Personal data
- Unverified accusations
- Production-only configuration
